3+**程序**：程序是指令、数据及其代码组织形式的描述

**进程**：进程是程序在某个数据集合上的一次执行过程。是系统进行资源分配的调度的基本单位，是程序的一个具体体现，是线程的容器

**线程**：线程是操作系统能够进行运算调度的最小单位，是进程中的实际运作单位
