# 储存器分类：

储存器分为两列，易失性和非易失性。

+ 易失性
  + 目前大多数是作为内存使用，因为速度快。但是掉电数据丢失
+ 非易失性
  + 目前大多数是作为rom、硬盘来使用。相比速度没那么快。但是掉电数据保存
  + 那么哪些数据需要掉电保存的呢？
    + 数据信息、图片、视频
    + 固件，程序、出场固件、wifi固件之类的
    + 字库数据
    + 设定数据
